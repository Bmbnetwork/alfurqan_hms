Alfurqan Hospital Management System (AHMS)

Alfurqan Hospital Management System (AHMS) is a modern, web-based Hospital Management System developed as a university project using Alfurqan Clinic & Maternity Limited, Bauchi as the case study. The system is designed to digitize daily hospital operations by replacing manual record-keeping with a secure, efficient, and user-friendly platform.

Built with PHP, MySQL, HTML5, CSS3, JavaScript, and XAMPP, AHMS streamlines patient registration, appointment scheduling, clinical consultations, antenatal care management, pharmacy operations, laboratory reporting, billing, and electronic medical records through dedicated role-based dashboards.

✨ Key Features
🔐 Secure multi-role authentication (Admin, Doctor, Nurse, Pharmacist, Laboratory Attendant, and Patient)
🏥 Role-based dashboards with unique UI/UX for each user
👤 Patient registration and electronic medical records (EMR)
📅 Online appointment booking and appointment management
👨‍⚕️ Doctor consultation and diagnosis management
🤰 Antenatal (ANC) monitoring and pregnancy record management
❤️ Nurse dashboard for recording patient vital signs and monitoring
💊 Pharmacy inventory, prescription fulfillment, stock management, and drug sales
🧪 Laboratory test requests, result entry, and report management
💳 Billing and payment management
🔍 Fast patient search using Patient ID
📊 Administrative dashboard with hospital statistics and operational overview
🗂️ Centralized database for efficient data storage and retrieval
🛠️ Technology Stack
Frontend: HTML5, CSS3, JavaScript
Backend: PHP
Database: MySQL
Local Server: XAMPP
🎯 Project Goal

The primary objective of AHMS is to improve healthcare service delivery by digitizing hospital workflows, reducing paperwork, enhancing patient record management, and improving collaboration among different hospital departments through an integrated web-based platform.

This project demonstrates practical application of web development technologies in healthcare information systems and serves as a scalable foundation for future enhancements such as AI-powered clinical decision support, online consultations, SMS/email notifications, inventory analytics, and cloud deployment.

Note: This project was developed for academic purposes using Alfurqan Clinic & Maternity Limited, Bauchi as the case study. It is designed to simulate real-world hospital operations while remaining simple enough for educational implementation and demonstration.
